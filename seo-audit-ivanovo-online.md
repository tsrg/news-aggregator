# SEO Audit: ivanovo.online
**Date:** 8 апреля 2026  
**Тип сайта:** Региональный новостной портал (Ивановская область)  
**Фреймворк:** Nuxt.js (SSR)

---

## Краткое резюме

Сайт ivanovo.online представляет собой региональный новостной агрегатор для Ивановской области. Проект технически реализован на современном стеке (Nuxt.js SSR), однако **полностью отсутствует в индексе Google и Яндекса** — поиск `site:ivanovo.online` не возвращает ни одного результата. Это критическая проблема, требующая немедленного устранения. Конкуренты (ivanovonews.ru, newsivanovo.ru, aif.ru, mkivanovo.ru) уверенно занимают топ по ключевым запросам, а сам сайт не виден ни по одному из них.

### Топ-5 приоритетных проблем

1. 🔴 **Нулевая индексация** — сайт отсутствует в результатах поиска Google и Яндекс
2. 🔴 **Ссылка на localhost в production-конфиге** — `apiBase:"http://localhost:3002"` публично видна в HTML
3. 🔴 **Sitemap содержит только 14 статических страниц** — ни одна новостная статья в него не включена
4. 🔴 **Нечитаемые URL статей** — `/news/cmnq4uf1o000310giipniavd0` вместо `/news/prokuratura-yuryevets-zemlya`
5. 🟠 **Отсутствие alt-текстов** у всех изображений на сайте

### 1.3 Sitemap

**URL:** `https://ivanovo.online/sitemap.xml`

| Параметр | Статус |
|---|---|
| Файл доступен | ✅ |
| Ссылка в robots.txt | ✅ |
| Количество URL | 🔴 **Только 14 URL** |
| Включены статьи/новости | 🔴 **Нет** |
| Обновляется при добавлении контента | 🔴 Не определено |
| Структура URL в sitemap | ✅ Валидная |

**Критическая проблема:** Sitemap содержит только 14 статических страниц (разделы, About, Contacts и т.д.). **Ни одна новостная статья не включена в sitemap.** Поисковые роботы не получают сигнала о существовании конкретных материалов. Для новостного сайта это означает, что свежий контент может вообще не попасть в индекс.

**Рекомендации:**
- Реализовать **динамический sitemap** со всеми статьями (`/news/[slug]`)
- Добавить **News Sitemap** (`<news:news>` элементы) для попадания в Google News
- Обновлять sitemap автоматически при публикации каждой статьи
- Использовать `changefreq: always` для новостных страниц

### 1.4 Рендеринг (SSR vs CSR)

**Фреймворк:** Nuxt.js (Vue.js) — **SSR правильно реализован**

✅ Контент присутствует в HTML без выполнения JavaScript  
✅ JSON-LD schema embed-ится на сервере  
✅ Поисковые боты могут читать контент

⚠️ **Обнаружена потенциальная проблема:**  
В исходном коде страницы публично видна конфигурация:
```
window.__NUXT__.config = { public: { apiBase: "http://localhost:3002" } }
```
Это означает, что **клиентские API-запросы** (навигация внутри SPA, бесконечная прокрутка, загрузка дополнительных новостей) идут на `localhost:3002` — адрес, недоступный из браузера пользователей. Это вероятно вызывает ошибки при SPA-навигации и отображении контента после первичной загрузки.

**Рекомендации:**
- Задать корректный публичный API URL через переменную окружения `NUXT_PUBLIC_API_BASE=https://api.ivanovo.online` (или аналог)
- Убедиться, что `publicRuntimeConfig` не раскрывает внутренние адреса инфраструктуры

**Рекомендация:** Перейти на SEO-friendly слаги:
- ❌ `/news/cmnq4uf1o000310giipniavd0`
- ✅ `/news/prokuratura-dobyvaetsya-vozrata-zemli-yuryevets`

### 2.4 Изображения

| Параметр | Статус |
|---|---|

| Форматы изображений | ℹ️ PNG (стоит рассмотреть WebP) |
| Lazy loading | ✅ Предположительно реализован |

**Критичность:** Отсутствие alt-текстов — двойная проблема:
1. **SEO**: Поисковики не понимают, что изображено
2. **Доступность**: Нарушает WCAG стандарты

**Рекомендация:** Добавить осмысленные alt-атрибуты к каждому изображению, например: `alt="Европейские лесные коты в Ивановском зоопарке"`

   — Вручную добавить главную, разделы, 10–20 ключевых статей

5. **Включить alt-текст для всех изображений**
   — Автоматически подставлять заголовок статьи как alt-текст

   — Например: `/news/prokuratura-yuryevets-zemlya-2026-04`
   — Настроить 301-редиректы со старых URL

   — Для попадания в Google News и Яндекс.Новости

8. **Исправить мета-описания** — расширить до 140–160 символов

9. **Оптимизировать H1 главной и разделов** — включить ключевые запросы

### 🟡 Средний приоритет (1–2 месяца)

10. **Добавить именованных авторов** к статьям + страницы авторов

11. **Увеличить объём статей** до 500+ слов

12. **Начать линкбилдинг** — первые упоминания в местных сообществах, соцсетях, пресс-релизы

14. **Оптимизировать title разделов** с конкретными ключевыми запросами

### ℹ️ Долгосрочно

15. Создать тематические кластеры контента по ключевым темам региона
